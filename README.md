# Junior Cybersecurity Portfolio

This repository contains hands-on cybersecurity labs and academic projects completed as part of my BSc (Hons) Cyber Security programme.

The portfolio demonstrates practical experience in:
- Ethical hacking and reconnaissance
- Network design and configuration
- Network traffic analysis using PCAP files
- Basic vulnerability identification and documentation
- Web application testing fundamentals

## Contents
- Ethical Hacking Lab
- Network Design (Cisco Packet Tracer)
- Network Traffic Analysis (Wireshark)
- Web Testing Fundamentals

All work was completed in controlled lab environments for educational purposes.
