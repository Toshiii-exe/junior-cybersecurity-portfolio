# Web Testing Fundamentals

This project introduces basic web application testing concepts using JavaScript and Cypress.io, focusing on user flows and functional behaviour.

## Focus Areas
- Understanding common user interactions
- Writing basic end-to-end test scenarios
- Identifying functional and usability issues

## Tools Used
- JavaScript
- Cypress.io

## Key Learning Outcomes
- Appreciating the role of automated testing in software quality
- Understanding how testing supports secure and reliable applications
