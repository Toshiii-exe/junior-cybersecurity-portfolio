# Ethical Hacking Lab

This project focuses on reconnaissance and enumeration techniques used during the early stages of ethical hacking.

## Objectives
- Perform reconnaissance on a target system
- Identify open ports and exposed services
- Analyse potential attack surfaces
- Document findings and mitigation suggestions

## Tools Used
- Nmap
- Linux (Ubuntu)
- Virtual lab environment

## Key Learning Outcomes
- Understanding the role of reconnaissance in ethical hacking
- Interpreting Nmap scan results
- Identifying potential vulnerabilities responsibly
